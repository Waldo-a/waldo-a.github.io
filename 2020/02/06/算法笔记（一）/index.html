#define 标识符 任何语句或片段 例如 #define ADD(a,b) ((a) + (b))

条件运算符 （ ？ ： ）是c语言中唯一一个三目运算符。

输入输出

scanf中， 数组名称本身就代表了这个数组第一个元素的地址，所以不用再加取地址运算符

除了%c外，scanf对其他格式符的输入是以空白符（空格、换行）为结束标志的。

字符数组使用%s读入的时候以空格跟换行为读入结束的标志

scanf的%c格式是可以读入空格和换行的

double类型的变量，其输出格式为%f，而在scanf中却是%lf

%md可以使不足m位的int型变量以m位进行右对齐输出，其中高位用空格补齐，如果变量本身超过m位，则保持原样

%0md，当变量不足m位时，讲在前面补足够数量的0而不是空格

%.mf可以让浮点数保留m位小数输出，这个“保留”使用的时精度的“四舍六入五成双”规则，题目中要求输出保留xx位小数，就是按照这个格式输出（如果时四舍五入，需要用round函数）

getchar输入单个字符，putchar输出单个字符（可识别换行符）

typedef long long LL; //给复杂的数据类型起别名，提高编码效率

常用math函数

  1. fabs(double x) 对double类型变量取绝对值
  2. floor(double x) 和 ceil(double x) 分辨用于double型变量向下取整和向上取整，返回类型为double
  3. pow(double r, double p) 返回 r^p
  4. sqrt(double x) 返回double型变量的算术平方根
  5. log(double x) 返回以自然对数位底的对数
  6. sin() cos() tan() pi = acos(-1.0)
  7. asin() acos() atan 返回反正弦、反余弦、反正切值
  8. round(double x) 将double型变量四舍五入，返回类型也是double，需进行取整

选择结构

只有确定不会出错情况下，才将大括号舍去

if(n != 0) 等价于 if(n)

if(n == 0) 等价于 if(!n)

数组

如果数组大小较大（大概10^6级别），则需要将其定义在主函数外面，否则程序异常退出，原因是函数内部申请的局部变量来自系统栈，允许申请的空间小。而函数外申请的全局变量来自静态存储区，允许申请的空间较大。

其他

memset（数组名，值，sizeof(数组名)); 按字节赋值，建议只赋0，-1。其他的用fill

字符数组可以通过直接赋值字符串来初始化（仅限初始化，程序其他地方不允许直接赋值整个字符串）

字符数组的 输入输出

使用%s时，不用加&

输入与输出相同

gets用来输入一行字符串（gets识别换行符\n作为输入结束，因此scanf整数后，想要使用gets，需要先用getchar接受整数后的换行符），并将其存放在一维数组中

字符数组的末尾都有一个空字符\0，以表示存放的字符串的结尾。\0在使用gets或scanf输入字符时会自动添加在输入的字符串后面，并占用一个字符位，而puts与printf就是通过识别\0作为字符串的结尾来输出的。

特别提醒，如果不是使用scanf和gets来输入一个字符串，一定要在输入的每个字符串后加”\0”，否则输出乱码。

string头文件

  1. strlen 返回字符数组中第一个\0的字符前的个数
  2. strcmp 按字典序返回 负整数、0、正整数
  3. strcpy(str1, str2) 把str2 复制给str1
  4. strcat(str1,str2) 把str2接在str1后面
  5. sscanf(str, “%d”, &n) 把screen的内容以%d格式传输到n中（自左向右）
  6. sprintf(str, “%d”, n); 把n以“%d”格式写到str字符数组中（自右向左）

复杂度

一般的OJ系统，一秒能承受的运算次数大约10^7 ~ 10^8。

因此O(n^2)的算法当n的规模为1000时可以承受，而n=100000时不可承受
